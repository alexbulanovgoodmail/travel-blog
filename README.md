## Требования

- Node.js: ^20.19.0 || >=22.12.0

## Начало работы

1. Установка зависимостей:

```sh
npm install
```

2. Запуск сервера разработки:

```sh
npm run dev
```

## Доступные команды

### Разработка

- `npm run dev` - запуск сервера разработки
- `npm run preview` - предварительный просмотр собранного проекта

### Сборка

- `npm run build` - полная сборка проекта с проверкой типов
- `npm run build-only` - только сборка проекта без проверки типов
- `npm run type-check` - проверка типов TypeScript

### Линтинг и форматирование

- `npm run lint` - проверка стилей и скриптов
- `npm run lint:styles` - проверка стилей
- `npm run lint:styles:fix` - исправление стилей
- `npm run lint:scripts` - проверка скриптов
- `npm run lint:scripts:fix` - исправление скриптов
- `npm run format` - проверка форматирования
- `npm run format:fix` - исправление форматирования

## Конфигурация

Для настройки проекта обратитесь к следующим файлам:

- `vite.config.ts` - конфигурация Vite
- `tsconfig.json` - настройки TypeScript
- `eslint.config.ts` - правила ESLint
- `stylelint.config.mjs` - правила Stylelint
